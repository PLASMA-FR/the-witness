# Demo script

1. Run `the-witness start` and show setup wizard.
2. Select Ollama/Gemma 4 E4B or Demo if no local model is installed.
3. Add a watched endpoint.
4. Send a bad answer through the proxy.
5. Show LiteRT prefilter/risk indicator, Gemma judge rejection, prompt repair, retry, approval, and JSONL audit log.
6. Open Model Manager and show Fine-tuned Witness Gemma 4 E2B/E4B Judge choices, Colab training path, and optional Kaggle download command.
